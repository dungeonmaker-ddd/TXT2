# 🧩 首页模块共享组件库设计文档 v1.0

> **基于后端配置驱动的通用组件库 - 为所有页面提供可复用的标准化组件**

---

## 📋 **文档信息**

| 项目信息 | 详细内容 |
|---------|---------|
| **文档名称** | 首页模块共享组件库 |
| **设计版本** | v1.0 |
| **创建时间** | 2025年9月 |
| **设计理念** | 后端配置驱动 + 组件复用最大化 |
| **适用范围** | 首页模块所有页面 |
| **配置策略** | 所有样式、行为、数据通过后端API配置 |

---

## 🎯 **设计原则**

### 📡 **后端配置驱动原则**
- ✅ **组件外观完全由后端配置控制**：颜色、尺寸、圆角、间距等
- ✅ **组件行为通过后端定义**：动画效果、交互反馈、状态变化
- ✅ **组件内容由后端数据提供**：文本、图标、布局结构
- ✅ **组件变体通过后端类型字段区分**：主要/次要/特殊等变体

### 🧩 **组件复用最大化原则**
- ✅ **单一组件支持多种场景**：通过配置参数适配不同使用场景
- ✅ **组件功能模块化**：将复杂组件拆分为可组合的功能模块
- ✅ **组件状态标准化**：统一的加载、错误、成功、空状态处理
- ✅ **组件接口一致性**：相同类型组件使用统一的数据接口

### ⚛️ **1. 基础交互组件 (Basic Interaction)**

#### 🔘 **UniversalButton - 通用按钮组件**

```typescript

{
  buttonConfig: {
    variant: "primary" | "secondary" | "filter" | "tab" | "icon",
    size: "small" | "medium" | "large",
    shape: "rectangle" | "rounded" | "circle",
    state: "default" | "selected" | "disabled" | "loading",
    
    // 外观配置
    appearance: {
      backgroundColor: string,
      textColor: string,
      borderColor: string,
      borderWidth: number,
      borderRadius: number,
      fontSize: number,
      fontWeight: string,
    },
    
    // 行为配置
    behavior: {
      animationDuration: number,
      pressScale: number,
      hapticFeedback: boolean,
      debounceTime: number,
    },
    
    // 内容配置

      text?: string,
      icon?: string,
      iconPosition: "left" | "right" | "only",
      badge?: {
        count: number,

【使用场景映射】
- 首页搜索按钮：variant="primary", size="medium"
```

## 🔄 **组件状态管理体系**

### 📊 **统一状态规范**

#### 🔄 **加载状态 (Loading States)**
```typescript
loadingState: {
  variant: "skeleton" | "spinner" | "shimmer" | "pulse",
  duration: number,
  text?: string,
  overlay: boolean,
}

【应用场景】

- 闪光效果：图片懒加载时的占位动画
- 脉冲效果：实时数据更新时的视觉反馈
```

#### ❌ **错误状态 (Error States)**
```typescript
errorState: {
  variant: "inline" | "overlay" | "toast" | "banner",

  retryEnabled: boolean,

【应用场景】

- 操作错误：表单验证失败的字段错误
- 系统错误：全局错误覆盖层
```

#### 📊 **配置缓存策略**
```typescript
缓存层级：
1. 内存缓存：热点组件配置，缓存5分钟
2. 本地存储：基础配置，缓存24小时
3. CDN缓存：静态资源配置，缓存7天
4. 数据库：完整配置数据，永久存储

缓存更新：
- 增量更新：只传输变更的配置项
- 版本比较：客户端版本号对比更新
- 预加载：下一页面配置预先获取
- 降级机制：网络异常时使用本地缓存
```
